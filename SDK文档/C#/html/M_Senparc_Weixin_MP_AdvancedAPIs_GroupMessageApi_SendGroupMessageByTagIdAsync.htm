参数

accessTokenOrAppId
类型：SystemString
AccessToken或AppId（推荐使用AppId，需要先注册）
tagId
类型：SystemString
群发到的标签的tag_id，若is_to_all值为true，可不填写tag_id
value
类型：SystemString
群发媒体文件时传入mediaId,群发文本消息时传入content,群发卡券时传入cardId
type
类型：Senparc.Weixin.MPGroupMessageType
isToAll (Optional)
类型：SystemBoolean
用于设定是否向全部用户发送，值为true或false，选择true该消息群发给所有用户，选择false可根据group_id发送给指定群组的用户
sendIgnoreReprint (Optional)
类型：SystemBoolean
待群发的文章被判定为转载时，是否继续群发
clientmsgid (Optional)
类型：SystemString
开发者侧群发msgid，长度限制64字节，如不填，则后台默认以群发范围和群发内容的摘要值做为clientmsgid
timeOut (Optional)
类型：SystemInt32
代理请求超时时间（毫秒）

返回值

类型：TaskSendResult
