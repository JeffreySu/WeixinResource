参数

accessTokenOrAppKey
类型： SystemString
调用接口凭证（AccessToken）或AppKey（根据AccessTokenContainer.BuildingKey(corpId, corpSecret)方法获得）
departmentId
类型： SystemInt32
获取的部门id
fetchChild
类型： SystemInt32
1/0：是否递归获取子部门下面的成员
status
类型： SystemInt32
0获取全部员工，1获取已关注成员列表，2获取禁用成员列表，4获取未关注成员列表。status可叠加
maxJsonLength (Optional)
类型： SystemNullableInt32
设置 JavaScriptSerializer 类接受的 JSON 字符串的最大长度

返回值

类型： GetDepartmentMemberResult
备注
2016-04-16：Zeje添加参数maxJsonLength：企业号通讯录扩容后，存在Json长度不够的情况。
