参数

serviceProvider
类型：SystemIServiceProvider
.NetCore 下的服务器提供程序，如果 .NET Framework 则保留 null
url
类型：SystemString
请求Url
cookieContainer (Optional)
类型：System.NetCookieContainer
CookieContainer，如果不需要则设为null
fileDictionary (Optional)
类型：System.Collections.GenericDictionaryString, String
需要Post的文件（Dictionary 的 Key=name，Value=绝对路径）
postDataDictionary (Optional)
类型：System.Collections.GenericDictionaryString, String
需要Post的键值对（name,value）
encoding (Optional)
类型：System.TextEncoding
cer (Optional)
类型：System.Security.Cryptography.X509CertificatesX509Certificate2
证书，如果不需要则保留null
useAjax (Optional)
类型：SystemBoolean
afterReturnText (Optional)
类型：SystemActionString, String
返回JSON本文，并在进行序列化之前触发，参数分别为：url、returnText
timeOut (Optional)
类型：SystemInt32
代理请求超时时间（毫秒）
