Public Shared Function SendTextCardAsync ( 
	accessTokenOrAppKey As String,
	agentId As String,
	title As String,
	description As String,
	url As String,
	Optional btntxt As String = Nothing,
	Optional toUser As String = Nothing,
	Optional toParty As String = Nothing,
	Optional toTag As String = Nothing,
	Optional timeOut As Integer = 10000
) As Task(Of MassResult)

参数

accessTokenOrAppKey
类型：SystemString
调用接口凭证（AccessToken）或AppKey（根据AccessTokenContainer.BuildingKey(corpId, corpSecret)方法获得）
agentId
类型：SystemString
企业应用的id，可在应用的设置页面查看
title
类型：SystemString
标题，不超过128个字节，超过会自动截断
description
类型：SystemString
描述，不超过512个字节，超过会自动截断
url
类型：SystemString
点击后跳转的链接
btntxt (Optional)
类型：SystemString
按钮文字。 默认为“详情”， 不超过4个文字，超过自动截断。
toUser (Optional)
类型：SystemString
UserID列表（消息接收者，多个接收者用‘|’分隔）。特殊情况：指定为@all，则向关注该企业应用的全部成员发送
toParty (Optional)
类型：SystemString
PartyID列表，多个接受者用‘|’分隔。当touser为@all时忽略本参数
toTag (Optional)
类型：SystemString
TagID列表，多个接受者用‘|’分隔。当touser为@all时忽略本参数
timeOut (Optional)
类型：SystemInt32
代理请求超时时间（毫秒）

返回值

类型：TaskMassResult
