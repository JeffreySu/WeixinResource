Config 类型公开以下成员。

属性
  名称说明
ApiMpFileHost
公众号（小程序）、开放平台【文件下载】 API 的服务器地址（默认为：http://file.api.weixin.qq.com）
ApiMpHost
公众号（小程序）、开放平台 API 的服务器地址（默认为：https://api.weixin.qq.com）
ApiWorkHost
企业微信API的服务器地址（默认为：https://qyapi.weixin.qq.com）
DefaultCacheNamespace
默认缓存键的第一级命名空间，默认值：DefaultCache
DefaultSenparcWeixinSetting 已过时。
默认微信配置
IsDebug

指定是否是Debug状态，如果是，系统会自动输出日志。

如果 CO2NET.Config.IsDebug 为 true，则此参数也会为 true，否则以此参数为准。

RootDictionaryPath
网站根目录绝对路径
SenparcWeixinSetting

微信全局配置

注意：在程序运行过程中修改 SenparcWeixinSetting.Items 中的微信配置值，并不能修改 Container 中的对应信息（如AppSecret），

如果需要修改微信信息（如AppSecret）应该使用 xxContainer.Register() 修改，这里的值也会随之更新。

ThrownWhenJsonResultFaild
当 JsonResult 不为“成功”状态时，是否抛出异常，默认为 true
UseSandBoxPay
微信支付使用沙箱模式（默认为false）
