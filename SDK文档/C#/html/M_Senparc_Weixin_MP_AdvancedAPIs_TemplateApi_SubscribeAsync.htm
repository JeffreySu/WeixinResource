Public Shared Function SubscribeAsync (

参数

accessTokenOrAppId
类型：SystemString
AccessToken或AppId（推荐使用AppId，需要先注册）
toUserOpenId
类型：SystemString
填接收消息的用户openid
templateId
类型：SystemString
订阅消息模板ID
scene
类型：SystemString
订阅场景值
title
类型：SystemString
消息标题，15字以内
data
类型：SystemObject
消息正文，value为消息内容，color为颜色，200字以内
url (Optional)
类型：SystemString
点击消息跳转的链接，需要有ICP备案
timeOut (Optional)
类型：SystemInt32

类型：TaskWxJsonResult
