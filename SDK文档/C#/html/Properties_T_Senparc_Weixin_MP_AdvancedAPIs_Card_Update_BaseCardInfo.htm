Update_BaseCardInfo 类型公开以下成员。

属性
  名称说明
can_give_friend
卡券是否可转赠，填写true 或false,true 代表可转赠。默认false。 不需提审
can_share
领取卡券原生页面是否可分享，填写true 或false，true 代表可分享。默认false。 不需提审
center_app_brand_pass
自定义居中使用入口小程序页面地址
center_app_brand_user_name
显示在顶部居中的自定义跳转小程序的user_name 格式为原始id+@app 小程序原始id可以在小程序的设置页面底部查看到
center_sub_title
显示在顶部居中的自定义cell入口右侧的提示语 非必填
center_title
顶部居中的自定义cell入口名称 非必填
center_url
顶部居中的自定义cell入口跳转外链的地址链接 非必填
color
券颜色。按色彩规范标注填写Color010-Color100 需提审
custom_app_brand_pass
自定义使用入口小程序页面地址
custom_app_brand_user_name
自定义使用入口跳转小程序的user_name，格式为小程序原始id+@app 小程序原始id可以在小程序的设置页面底部查看到
custom_url
商户自定义url 地址，支持卡券页内跳转,跳转页面内容需与自定义cell 名称保持一致。 不需提审
custom_url_name
自定义跳转入口的名字。 string（16）不需提审
custom_url_sub_title
显示在入口右侧的提示语 string（18） 不需提审
date_info
使用日期，有效期的信息 必填
description
使用说明。长文本描述，可以分行，上限为1000 个汉字。 需提审
get_limit
每人最大领取次数，不填写默认等于quantity。 不需提审
location_id_list
门店位置ID。商户需在mp 平台上录入门店信息或调用批量导入门店信息接口获取门店位置ID。 不需提审
logo_url
卡券的商户logo，尺寸为300*300。 需提审
notice
使用提醒，字数上限为9 个汉字。（一句话描述，展示在首页，示例：请出示二维码核销卡券） 需提审
promotion_app_brand_pass
自定义营销入口小程序页面地址
promotion_app_brand_user_name
小程序的user_name 格式为小程序原始id+@app 小程序原始id可以在小程序的设置页面底部查看到
promotion_url
入口跳转外链的地址链接 不需提审
promotion_url_name
营销场景的自定义入口名称 string（16）不需提审
promotion_url_sub_title
显示在营销入口右侧的提示语。 不需提审
service_phone
客服电话 不需提审
