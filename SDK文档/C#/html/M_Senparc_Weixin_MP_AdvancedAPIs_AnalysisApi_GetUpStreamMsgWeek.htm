	string accessTokenOrAppId,
	string beginDate,

Public Shared Function GetUpStreamMsgWeek ( 
	accessTokenOrAppId As String,
	beginDate As String,
	endDate As String,
	Optional timeOut As Integer = 10000
) As AnalysisResultJson(Of UpStreamMsgWeekItem)
参数
accessTokenOrAppId
类型：SystemString
AccessToken或AppId（推荐使用AppId，需要先注册）
beginDate
类型：SystemString
获取数据的起始日期，begin_date和end_date的差值需小于“最大时间跨度”（比如最大时间跨度为1时，begin_date和end_date的差值只能为0，才能小于1），否则会报错
endDate
类型：SystemString
获取数据的结束日期，end_date允许设置的最大值为昨日
timeOut (Optional)
类型：SystemInt32
代理请求超时时间（毫秒）
返回值
类型：AnalysisResultJsonUpStreamMsgWeekItem
